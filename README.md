Smart Campus Map
Interactive campus navigation web app built with HTML/CSS/JS. Explore departments, hostels, facilities, routes, events, and more on an interactive map.
✨ Features
	•	Campus Overview - Quick stats (25+ buildings, 10+ hostels, library, medical center)
	•	Department Info - Click buildings for courses, labs, faculty contacts
	•	Facilities - Canteens, gym, library, ATM with timings/services
	•	Hostel Details - Capacity, WiFi, warden contacts
	•	Visitor Guide - Entrances, parking, admission office
	•	Important Routes - Shortest paths (Gate→Admin, Library→Hostel)
	•	Live Events - Tech fests, workshops, sports marked on map
	•	Food Spots - Cafeteria, juice corners, study areas
	•	Emergency Points - Medical, security, fire safety
	•	Quick Search - Search buildings, departments, facilities
	•	3-Step Explore - Search → Click → Get Directions
